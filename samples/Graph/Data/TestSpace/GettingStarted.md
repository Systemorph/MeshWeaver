# Getting Started

This page exists so there is *something* concrete to talk to an agent about.

## A few facts to reference

- The Test Space is a sandbox; nothing here is real data.
- It has three pages: this one, [Sample Notes](/TestSpace/SampleNotes), and a nested [Setup Guide](/TestSpace/Guides/Setup).
- The space is intended for verifying the chat header, the context chip, the side-panel peek, and the thread navigation.

## Try this

> Open a chat from this page and ask: *"Summarise the Test Space and list its pages."*

The agent should pick up this page as the conversation's context (you'll see it as a chip in the composer and in the thread header).
