# Test Space

A deliberately small **sandbox** for trying out MeshWeaver's chat and thread UX. Navigate here, attach it as the **context** of a conversation, and start threads against its pages.

## Pages

| Page | What it's for |
|------|---------------|
| [Getting Started](/TestSpace/GettingStarted) | The first thing to read — what's here and how to use it |
| [Sample Notes](/TestSpace/SampleNotes) | A page with some throwaway content to reference in chats |
| [Setup Guide](/TestSpace/Guides/Setup) | A nested page (under `Guides/`) for testing hierarchy |

## How to use this for testing

1. Open any page and start a chat — the page becomes the **context** of the thread.
2. Use the context chip in the composer (and the thread header) to peek back at the context object.
3. Create a few threads so the "my other open threads" navigation has something to cycle through.
