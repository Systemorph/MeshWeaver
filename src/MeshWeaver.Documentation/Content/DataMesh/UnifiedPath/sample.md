### Sample embedded markdown

This file lives in the **`content`** collection of the *Unified Path* node and is embedded inline
via `@@content/sample.md`. Files like this ship inside the documentation assembly and are mapped —
read-only — as each doc node's own `content` collection, so the embed renders straight from the
shipped asset with no copy and works offline. See
[Authoring Documentation](@/Doc/Architecture/AuthoringDocumentation).
