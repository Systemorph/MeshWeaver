# AI models now show their provider's logo

Picking a model is easier to scan: every model now carries its maker's logo — Anthropic, OpenAI, Google, Mistral, Meta, DeepSeek, xAI, Qwen, Perplexity, GitHub Copilot and more — in place of the identical sparkle every model used to show.

The logo is inferred from the model itself, so a Claude model served through Azure or an OpenRouter gateway still reads as Anthropic. Models from a provider we don't yet recognise keep the neutral sparkle.
